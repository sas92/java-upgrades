# Java Upgrades (Java 8 → Java 25)

This repository demonstrates the evolution of Java from version 8 through 25, highlighting key language features, APIs, and JVM improvements. Each version has its own folder under `examples/` with concise, runnable code samples.

## Structure
- `src/main/java/javaX/` → Feature demos for each Java release
- `docs/javaX.md` → Explanations, diagrams, and references

## How to Run
1. Clone the repo
2. Open in IntelliJ IDEA
3. Use Gradle to build and run examples

## Goals
- Provide clear, runnable examples
- Show practical use cases for new features
- Serve as a reference for developers upgrading legacy codebases

---

## Versions Covered
- Java 8: Lambdas, Streams, Optional, Date/Time API
- Java 9: Modules, JShell, Reactive Streams
- Java 10: `var`, Local Variable Type Inference
- Java 11: HTTP Client, String methods, `var` in lambda
- Java 12–25: Pattern Matching, Records, Sealed Classes, Virtual Threads, and more
